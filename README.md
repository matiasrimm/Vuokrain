# Vuokrain
A Java Spring Web App project for renting stuff over the internet.

Inspiration is some part gotten from Tori.fi - service.

# Features
- Hibernate JPA
- Hibernate Search
- Hibernate Validation
- Flyway Migrations
- Commons Fileupload
- Java Mail
- Spring Security
- Testing
  - Junit
  - Greenmail
  - Mockito
